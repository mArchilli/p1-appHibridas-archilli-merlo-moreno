Introducción

Nuestra API permite gestionar y obtener información sobre pacientes y sus datos asociados. Con ella, podras realizar diversas operaciones, como obtener información de pacientes individuales, listar todos los pacientes, obtener resultados paginados y ordenados, e incluso acceder a las sesiones relacionadas con cada paciente.

Pacientes

Haz una solicitud GET para obtener la lista completa de pacientes

GET /pacientes

Haz una solicitud GET para obtener la lista completa de pacientes ordenados por nombre de forma descendiente

GET pacientes/ordenamiento

Haz una solicitud GET para obtener la lista completa de pacientes paginados

GET pacientes/paginacion

Adicionalmente podrás obtener información de un paciente específico mediante su ID utilizando la ruta /paciente/:id. Si quieres filtrar pacientes por nombre podrás hacerlo mediante la ruta /paciente/:nombre. En ocasiones necesitarás otros datos como la cobertura médica, podrás obtenerlo en la ruta /paciente/ cobertura_medica/:cobertura_medica o si quisieras filtrar por género deberas ingresar a la ruta /paciente/generos/:genero

Sesiones

Haz una solicitud GET para obtener la lista completa de sesiones

GET /sesiones

Adicionalmente podrás obtener información de una sesión específica mediante su ID utilizando la ruta /sesiones/:id.

Notas y Consideraciones

Asegúrate de proporcionar datos correctos y completos al realizar solicitudes. La API utiliza validaciones para garantizar la precisión y seguridad de los datos. Ten en cuenta la seguridad y la privacidad de los datos de pacientes.

Cumple con las leyes y regulaciones aplicables al trabajar con información confidencial. Esperamos que esta introducción te ayude a entender las capacidades de nuestra API de Gestión de Pacientes. Si tienes preguntas o necesitas más información sobre cómo utilizar la API, ¡estamos aquí para ayudarte!
