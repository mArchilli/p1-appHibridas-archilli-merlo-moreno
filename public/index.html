The PsiCare API

¡Bienvenido a la API de Gestión de Pacientes! Nuestra API permite gestionar y obtener información sobre pacientes y sus datos asociados. Con ella, puedes realizar diversas operaciones, como obtener información de pacientes individuales, listar todos los pacientes, obtener resultados paginados y ordenados, e incluso acceder a las sesiones relacionadas con cada paciente.
